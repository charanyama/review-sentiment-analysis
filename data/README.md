# Data Directory

This folder contains datasets used for model training and evaluation.

## Dataset Information

The models in this repository are trained on the **IMDB Review Dataset**, a collection of 50,000 movie reviews with binary sentiment labels.

### Dataset Source

**Download:** [IMDB Dataset of 50K Movie Reviews](https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews)

**Platform:** Kaggle

**Size:** 50,000 reviews

**Labels:** Positive/Negative sentiment
